# Rust Atomics and Locks

**Low-Level Concurrency in Practice** by [Mara Bos](https://github.com/m-ou-se)

This is an mdBook version of the free online version of the book [Rust Atomics and Locks](https://mara.nl/atomics/).

## About the Book

Concurrency can be notoriously difficult to get right. This book provides a solid foundation for anyone working on Rust's concurrency primitives. You'll learn:

- Rust's atomics and memory ordering
- How threads communicate and synchronize
- Building your own locks, channels, and Arc
- Understanding the processor and operating system primitives
